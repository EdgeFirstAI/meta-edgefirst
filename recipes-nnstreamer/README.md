# recipes-nnstreamer

GStreamer/NNStreamer plugins for real-time ML inference on video and sensor streams.

- **nnstreamer** — EdgeFirst fork with dmabuf zero-copy, TFLite-VX CameraAdaptor, and Ara-2 sub-filter support
- **gst-edgefirst** — EdgeFirst GStreamer plugins (3D Spatial Perception, PointCloud, RadarCube, sensor fusion, visualizations, Zenoh bridges)
